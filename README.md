# VVCE Campus Map 

An interactive campus map for Vidyavardhaka College of Engineering (VVCE), Mysore.
Built with Leaflet.js and OpenStreetMap.

## Features

- Interactive map of the VVCE campus
- Live search for buildings and rooms
- Indoor floor plans with clickable rooms
- Outdoor and indoor route navigation
- Live GPS location (blue dot)

## Tech Stack

- HTML / CSS / JavaScript (vanilla)
- [Leaflet.js](https://leafletjs.com/) — interactive maps
- [OpenStreetMap](https://www.openstreetmap.org/) — map tiles
- Deployed on GitHub Pages


## Project Status

Prototype — currently includes buildings and Navigation stuff.
Full campus coverage in progress.
